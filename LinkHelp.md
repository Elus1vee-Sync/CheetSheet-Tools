
### Elevación de Privilegios y Post-Explotación
Recursos críticos para buscar configuraciones erróneas y escalar accesos en el sistema objetivo.

#### 🐧 Entornos Linux
*   **[GTFOBins](https://gtfobins.github.io/)** — Catálogo de binarios legítimos de Linux que pueden ser abusados para saltar restricciones locales de seguridad.
*   **[LinPEAS](https://github.com/peass-ng/PEASS-ng)** — El script de enumeración automatizada definitivo para encontrar rutas de escalada de privilegios en Linux.

#### 🪟 Entornos Windows
*   **[LOLBAS Project](https://lolbas-project.github.io/#)** — Equivalente a GTFOBins pero enfocado en binarios, librerías y scripts nativos de Windows (*Living Off The Land*).
*   **[WinPEAS](https://github.com/peass-ng/PEASS-ng)** — Script automatizado hermano de LinPEAS enfocado en la búsqueda de fallos y credenciales en Windows.
*   **[Seatbelt](https://github.com/GhostPack/Seatbelt)** — Herramienta orientada al *proceso de seguridad* para comprobar la configuración defensiva de un host Windows.
*   **[JAWS (Just Another Windows Escalation Script)](https://github.com/411Hall/JAWS)** — Script clásico en PowerShell diseñado para ayudar a identificar vectores potenciales de escalada en Windows.
